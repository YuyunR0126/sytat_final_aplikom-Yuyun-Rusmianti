LAPORAN PRAKTIKUM APLIKASI KOMPUTER

Hardware

Hardware merupakan perangkat keras pada komputer yang merupakan suatu komponen dari sebuah komputer yang berbentuk nyata, bisa dilihat untuk mendukung sistemkomputerisasi.Menurut James Oâ€™Brien Hardware adalah semua perlatan fisik yang digunakan dalam pemrosesan informasi, termasuk diantaranya CPU (Central Processing Unit), RAM (Random Access Memory), monitor, mouse, keyboard, printer, scanner, dan lain-lain.Secara umum fungsi utama dari hardware yaitu menerima input,mengolah data dan informasi seta menyimpan data
Berdasarkan fungsinya, perangkat keras komputer dibagi menjadi: Input Divice (Unit Masukan) : Perangkat keras komputer yang memiliki fungsi sebagai input atau masukan, baik itu perintah maupun koneksi ke dalam sistem komputer. Process Device (Unit Pemrosesan) : Perangkat keras komputer yang berhubungan dengan fungsi pemrosesan dalam komputer itu sendiri untuk menjalankan proses sistem komputer. Output Device (Unit Keluaran) : Perangkat keras komputer yang memberikan output/keluaran, baik berupa tampilan visual, suara, maupun tampilan cetak, dan lain sebagainya. Backing Storage (Unit Penyimpanan) : Peragkat keras komputer yang berfungsi sebagai media penyimpanan untuk menyimpan data-data komputer. Peripheral Device (Unit Periferal) : Perangkat keras komputer berupa unit tambahan yang digunakan sebagai pelengkap pada komputer.
Komponen dasar pada komputer terdiri dari input, process, output dan storage. Input device terdiri dari keyboard dan mouse, Process device adalah microprocessor (ALU, Internal Communication, Registers dan control section), Output device terdiri dari monitor dan printer, Storage external memory terdiri dari harddisk, Floppy drive, CD ROM, Magnetic tape. Storage internal memory terdiri dari RAM dan ROM. Sedangkan komponen Periferal Device merupakan komponen tambahan atau sebagai komponen yang belum ada atau tidak ada sebelumnya. Komponen Periferal ini contohnya : TV Tuner Card, Modem, Capture Card.

Sistem Operasi Windows

Sistem operasi adalah pengelola seluruh sumber-daya yang terdapat pada sistem komputer dan menyediakan sekumpulan layanan ke pemakai sehingga memudahkan dan menyamankan 234P dan penggunaan serta pemanfaatan sumber-daya sistem komputer. Sedangkan sistem operasi Windows adalah sistem operasi yang diciptakan oleh Microsoft, dimana sistem operasi ini menyediakan antarmuka grafis (GUI / Graphical User Interface) agar lebih mudah dioperasikan.Microsoft Windows atau lebih dikenal dengan sebutan Windows, Windows adalah keluarga sistem operasi komputer yang dikembangkan oleh Microsoft, dimana sistem operasi ini menyediakan antarmuka grafis (GUI / Graphical User Interface) agar lebih mudah dioperasikan.Virtualbox merupakan sebuah aplikasi virtual yang dapat menjalankan beberapa sistem operasi sekaligus dalam waktu yang bersamaan. Selain itu siapkan juga aplikasi Windows 10 iso untuk penginstalan Windows 10. Dengan bantuan Virtual Box dapat dimanfaatkan untuk untuk mengeksekusi sistem operasi tambahan di dalam sistem operasi utama. Virtual Box dapat bekerja di sistem operasi Windows, Linux, IBM, BSD, Mac OS X, dan lain sebagainya.

Sistem Operasi Linux

Linux adalah sebuah perangkat lunak yang berupa sistem operasi pada open source yang disebarluaskan secara gratis di bawah naungan lisensi GNU (Gnu Not Unix).Sistem operasi Linux pada awalya merupakan proyek hobi yang dibuat oleh Linus Torvalds yang terinspirasi oleh sistem operasi terdahulu yaitu MINIX (Mini UNIX) yang open source sehingga dalam perkembangannya telah melahirkan berbagai macam distro Linux (distribusi Linux) seperti Debian, Red Hat Linux, Slackware, Ubuntu dan lain sebagainya.Ubuntu adalah salah satu distribusi Linux yang berbasiskan Debian dan didistribusikan menjadi perangkat lunak sistem operasi yang bebas. Dengan kata lain, Ubuntu adalah pengembangan dari distro Linux Debian.

Microsoft Office Word

Microsoft Office adalah sebuah program aplikasi perkantoran yang dibuat oleh Microsoft untuk menunjang berbagai kegiatan perkantoran. Produk ini diciptakan untuk memudahkan pekerjaan kantor seperti pengetikan laporan, pembuatan laporan keuangan, pemaparan presentasi dan masih banyak lagi.Microsoft Office Word merupakan aplikasi perkantoran yang termasuk dalam paket Microsoft Office, merupakan perangkat lunak atau program pengolah kata andalan Microsoft.

Microsoft Office Excel dan PowerPoint

Microsoft Excel adalah sebuah program aplikasi lembar kerja spreadsheet yang dikhususkan untuk pengolahan angka. Microsoft Excel memiliki kemampuan untuk melakukan perhitungan dasar, menggunakan alat grafik, membuat tabel dan makro. Microsoft PowerPoint adalah sebuah program komputer untuk presentasi yang dikembangkan oleh Microsoft. 2. Didalam Microsoft Excel kita dapat membuat worksheet baru, memasukan data secara berurutan, menyisipkan baris baru/kolom baru, menggabungkan cell, melakukan perhitungan dengan formula Excel (SUM, MIN, MAX, AVERAGE, dll.) dan lain sebagainya.

Desain Grafis

Desain grafis merupakan suatu bentuk dari komunikasi visual yang memanfaatkan gambar sebagai media untuk menyampaikan pesan atau informasi seefektif mungkin.Dari sinilah Desain grafis juga seperti jenis desain lainnya merujuk kepada proses pembuatan, metode merancang, produk yang dihasilkan (rancangan), atau pun disiplin ilmu yang digunakan (desain).

Multimedia (Editing Video)

Multimedia adalah penggunaan komputer untuk menyajikan dan menggabungkan teks, suara, gambar, animasi, audio dan video dengan alat bantu (tool) dan koneksi (link) sehingga pengguna dapat melakukan navigasi, berinteraksi, berkarya dan berkomunikasi.Tujuan dari proses video editing adalah membuat suatu rekaman video mentah menjadi suatu tampilan video yang menarik dan lebih nyaman untuk dinikmati.

Pemrograman Web (HTML/CSS)

HTML adalah teknologi dasar yang digunakan untuk membuat dasar dari sebuah halaman web dengan menghubungkan teks/link antar halaman, memberi struktur dan membagikan informasi yang terkait dengan sebuah halaman web. Untuk mengelolah tampilan website dapat dilakukan dengan menambahkan CSS, namun tidak dengan sembarangan, kreatifitas developer juga sangat dibutuhkan untuk membuat suatu tampilan website yang menarik dan berkuaitas.HTML (HyperText Markup Language) adalah bahasa standar pemrograman untuk membuat suatu website yang bisa diakses dengan internet. Salah satu jenis HTML yaitu HTML 5 adalah sebuah prosedur pembuatan tampilan web baru yang merupakan penggabungan antara CSS, HTML itu sendiri . CSS (Cascading Style Sheet) adalah standar pembuatan dan pemakaian style untuk dokumen terstruktur. CSS digunakan untuk mempersingkat pemakaian tag HTML seperti halnya font, color, text, dan table menjadi lebih ringkas sehingga tidak terjadi pengulangan tulisan. Penggunaan yang paling umum digunakan adalah untuk menformat halaman web yang ditulis dalam tag tag HTML maupun XHTML. Fungsi html yang pertama adalah untuk membuat suatu halaman website yang dapat dibaca dan dipahami oleh pengguna dengan lebih mudah.

Pemrograman Web (Web Hosting)

Hosting (disebut juga Web Hosting / sewa hosting) adalah penyewaan tempat untuk menampung data-data yang diperlukan oleh sebuah website dan sehingga dapat diakses lewat Internet. Data disini dapat berupa file, gambar, email, aplikasi/program/script dan database.

Augmented Reality

Augmented reality (AR) adalah penggabungan antara objek virtual dengan objek nyata. Sebagai contoh adalah saat stasiun televisi yang menyiarkan pertandingan sepak bola, terdapat objek virtual, tentang skor pertandingan yang sedang berlangsung.Menurut Ronald Azuma pada tahun 1997, Augmented reality adalah menggabungkan dunia nyata dan virtual, bersifat interaktif secara real time, dan merupakan animasi 3D.Augmented reality merupakan perkembangan dari ditemukannya sebuah sensorama dengan visual, getaran dan bau, hingga meperkenalkan virtual reality di dunia maya. Setelah itu, berulah augmented reality mulai dikembangkan. Sampai saat ini penerapan augmented reality semakin membantu kehidupan masyarakat karena perkembangannya tersebut. Objek augmented reality dapat dibuat dengan menggunakan software Blender, sehingga dapat mebuat suatu objek 3D yang hampir sama dengan benda-benda di kehidupan nyata.
